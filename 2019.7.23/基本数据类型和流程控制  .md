### 值类型和引用类型的区别  
#### 值类型的变量直接在对战中保存变量的值，引用类型的变量在对战中保存的是对象的引用地址
#### 当把一个值类型的赋给另一个值类型的变量时，系统会在堆栈中保存两个完全相同的值；而把一个引用变量赋给另一个引用变量，在对战中的两个值虽然相同，但是由于这两个值都是堆中对象的引用地址，所以实际上引用的还是同一个对象
## 常量与变量  
### 1.常量   
#### C#常用 const 关键字声明变量 
```
const int pi =3.1415926;
```
#### 对程序进行编译的时候，编译器会把所有声明为 const 的常量全部替换为实际的常熟
### 2.变量  
#### 变量存储的值可能会发生变化，单变量名保持不变
### 3.匿名类型的局部变量
#### 如果是局部变量，在C#中还可以用var 来声明变量其类型。当开发人员记不清楚它到底是呢种数据类型时，使用var 声明其实非常方便，实际上采用这种方法声明依然是一种强类型的数据，只是具体的类型有编译器负责推断而已。
```
var key = Console.ReadKey();
```
#### 等价于  
```
ConsoleKeyInfo key = Console.ReadKey();
```
#### 显然，用var来声明要方便得多

### 运算符 
#### 点运算符：  
##### 指定类型或命名空间的成员，如System.Console.WriteLine("hello");
#### 圆括号运算符()：  
##### (1)指定表达式运算顺序；(2)用于显示转换；(3)用于方法或委托，即将参数放在括号内
#### 方括号运算符[]：  
##### (1)用于数组和索引  
##### (2)用于特性  
```

```
##### (3)用于指针(非托管模式)
```

{
    int [] nums = {0,1,2,3,4,5};
    fixed(int *p=nums)
    {
        p[0]=p[1]=1
        for(int i=2 ; i<100 ; i++)
        p[i]=p[i-1]+p[i-2];
    }
}
```
